🏠 GestiLoc — Gestion Immobilière Premium

GestiLoc est une application web élégante et intuitive conçue pour simplifier la gestion des locations immobilières. Développée pour LOKONON Paul, elle permet de suivre le patrimoine, les locataires, les flux financiers et les relances de loyers en un seul coup d'œil.

✨ Fonctionnalités

Tableau de bord dynamique : Visualisation instantanée des recettes mensuelles et du taux d'occupation.

Gestion du Patrimoine : Ajout et suivi des chambres ou appartements avec leurs tarifs respectifs.

Registre des Locataires : Gestion des contrats de bail et des informations de contact.

Flux Financiers : Suivi rigoureux des encaissements avec historique mensuel.

Système de Relance : Identification automatique des loyers impayés pour le mois en cours.

🚀 Comment utiliser ce projet ?

1. Installation

Il n'y a aucune installation complexe requise. Comme il s'agit d'une application "Single File", il vous suffit de télécharger le fichier gestiloc_v3_final.html et de l'ouvrir dans n'importe quel navigateur moderne.

2. Hébergement sur GitHub Pages

Pour transformer ce code en un site web accessible via un lien :

Renommez votre fichier principal en index.html (c'est impératif pour GitHub).

Allez dans les Settings de votre dépôt GitHub.

Cliquez sur Pages dans le menu de gauche.

Sous Branch, sélectionnez main (ou master) et cliquez sur Save.

Attendez 1 à 2 minutes : GitHub vous donnera un lien URL (ex: https://votre-nom.github.io/votre-depot/).
